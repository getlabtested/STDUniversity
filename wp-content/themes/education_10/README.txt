Education WordPress Theme
http://www.studiopress.com/themes/Education

INSTALL: 
1. Upload the theme folder via FTP to your wp-content/themes/ directory.
2. Go to your WordPress dashboard and select Appearance.
3. Select Education WordPress theme.
4. Download the latest version of the Featured Content Gallery plugin (http://www.featuredcontentgallery.com)
5. Upload the Featured Content Gallery plugin via FTP to your wp-content/plugins directory, and follow the directions from the plugin page to implement it.
6. Inside your WordPress dashboard, go to Appearance > Education Theme Options and configure them to your liking.

PSD'S:
If you would like to customize the header of your site, you can do so by downloading the Extras file from the theme download page in the support forum. Inside that you will find a folder of PSD files.

If you are looking for theme support, please visit http://www.studiopress.com/support
